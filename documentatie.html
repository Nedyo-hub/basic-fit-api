API Documentatie

Dit is een eenvoudige documentatie voor de gebruikers API.

Endpoints

  • GET /users: Haalt alle gebruikers op (met paginering via query parameters: ?limit=10&offset=0)
  • POST /users: Voegt een nieuwe gebruiker toe (vereist naam en e-mail)
  • GET /users/:id: Haalt een gebruiker op met een bepaald ID
  • PUT /users/:id: Werkt een gebruiker bij (vereist naam en e-mail)
  • DELETE /users/:id: Verwijdert een gebruiker met een bepaald ID

Opmerking: Alle velden moeten correct zijn ingevuld en voldoen aan de validaties (zoals geen cijfers in de naam, en een geldig e-mailadres).
